# 問題 5
テキストボックス 2 つ、コンボボックス 1 つ、ボタン 1 つを用意する。
コンボボックスの中身は、「+」、「-」、「*」、「/」の演算子が選択できるようになっている。
ボタンを押すと、テキストボックスの中身を数値とし、コンボボックスで選択した演算子により
計算した値をメッセージ表示する。
